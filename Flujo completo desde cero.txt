Flujo completo desde cero:
Paso 1: Descargar históricos completos (si no están ya)
python download_full_history.py
Genera: data_cache/*.parquet (48 símbolos)
Paso 2: Entrenar modelos de régimen
python train_regime_model.py
Lee: data_cache/*.parquet
Genera: regime_models/*.joblib + regime_models/regime_report.txt
Paso 3: Generar presets (lab_lite + lab_historico + extractor)
python pipeline.py --phase extractor --output-dir output/test_cluster_v2
Lee: data_cache/*.parquet, regime_models/*.joblib
Genera: output/test_cluster_v2/SYMBOL/presets_SYMBOL.csv, gemas_SYMBOL.csv, full_*.csv
Paso 4: Regime walk-forward (lo costoso, ~8h por símbolo)
python regime_walk_forward.py --presets-dir output/test_cluster_v2 --toxic-tail dynamic
Lee: data_cache/*.parquet, regime_models/*.joblib, output/test_cluster_v2/*/presets_*.csv
Genera: regime_wf/SYMBOL_specialist_configs.json, *_specialists_report.txt, *_cluster_K_specialists.csv
